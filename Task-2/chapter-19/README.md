# Chapter-19: Keeping Time, Scheduling Tasks, and Launching Programs

## Introduction
This Chapter explains how Python programs can track time, schedule tasks, and run automatically without user interaction. This chapter helped me understand how automation can be used to perform tasks at specific times or intervals.

## Explanation
This chapter explains the modules like time and datetime, which help programs read system time, pause execution, and calculate time differences. It explains concepts like Unix timestamps and how they are used to measure execution time
Through examples like building a stopwatch program, I learned how scripts can track elapsed time and respond to user input. The chapter also explains how programs can be scheduled to run automatically and how Python can launch other programs when needed.

## Implementation
In this Chapter , I created a simple program using the time module to display the current time repeatedly. The program runs a loop for a few seconds and prints the system time using __time.ctime()__ while pausing execution with __time.sleep()__.
This implementation helped me to  understand how programs track and control execution time. It also showed how developers use time-based logic in automation tasks like scheduling and monitoring processes.
