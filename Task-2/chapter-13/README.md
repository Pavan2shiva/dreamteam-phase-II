# Chapter - 13: Web Scraping

## Introduction
This Chapter mainly focuses on how Python programs can interact with the internet and collect useful data automatically. In this chapter, I learned how web scraping works and how developers use it to download and process web content instead of doing repetitive manual work online. It helped me understand how automation can save time when working with websites.

## Explanation
This chapter explains web scraping using Python modules like __webbrowser__, __requests__, and __BeautifulSoup__. It shows how programs can open web pages, download files from the internet, and extract useful data from HTML content. The chapter also explains basic concepts like URLs, HTTP vs HTTPS, and how APIs help programs communicate with web services. 
Through examples like opening maps automatically and downloading files from websites, I understood how automation reduces manual steps. The chapter also introduces error handling while downloading content and explains how JSON data from APIs can be processed in Python. 

In this chapter, I learned about important Python modules and how they are used to interact with the internet. I understood how modules like requests, webbrowser, and others help in downloading files, accessing web pages, and collecting useful data automatically. This chapter also showed how automation reduces manual effort and makes working with online resources much easier.

## Implementation
In this chapter, I created a simple weather finder program using Python and an online API. The program asks the user to enter a city name and then sends a request to the OpenWeather API using the requests module. After receiving the response, the program converts the JSON data into a dictionary and extracts useful details like temperature, humidity, and weather condition.

I converted temperature from Kelvin to Celsius and displayed the results clearly for the user. This implementation helped me understand how APIs work in real applications and how Python programs can collect live data from the internet automatically.

## observations
* Web scraping helps automate repetitive internet tasks.
* Using modules like requests makes downloading data simple.
* APIs are useful for structured and reliable data collection.
* Automation reduces manual effort while improving efficiency.
