# Task-1: Bandit & Git exercises

This task includes practice of both the Bandit wargame from OverTheWire and basic Git version control exercises. The Bandit section focuses on understanding Terminal command-line fundamentals, file handling, permissions. The Git section introduces essential version control workflows such as repository setup, tracking changes, committing updates, and managing project files.

Detailed steps, commands used, screenshots, and explanations of what I performed in each exercise are provided separately in the respective writeup.md files. This README serves only as a brief introduction to the work completed in Task-1.
